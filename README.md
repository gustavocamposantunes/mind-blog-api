# Mind Blog API

Projeto desenvolvido como parte de um teste técnico para uma vaga de estágio como desenvolvedor fullstack na MindGroup. 


# Tecnologias Utilizadas

* NestJS
* TypeORM
* Express
* Jest
* MySQL

# Funcionalidades

* Cadastro de novos usuários usuários
* Redefinição de senha
* Autenticação e autorização

## Installation

```bash
$ pnpm install
```

## Running the app

```bash
# development
$ pnpm run start

# watch mode
$ pnpm run start:dev

# production mode
$ pnpm run start:prod
```

## Test

```bash
# unit tests
$ pnpm run test

# e2e tests
$ pnpm run test:e2e

# test coverage
$ pnpm run test:cov
```
